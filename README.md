## Одностраничное приллжение на React

Приложение позволяет загрузить свой JSON файл, чтобы отслеживать свой прогресс в любых задачах, оставлять в них заметки и ставить дедлайны.

Чтобы воспользоваться приложением, скачайте репозиторий и выполните следующие команды:
 `npm install
 npm start`

Либо просто перейдите по ссылке: https://rin-rgb.github.io/Roadmap/
